# Node.py

A Python runtime which offers a module-solution similar to Node.js.

```python
this = require('that')
pathutils = require('./utils/path')

if require.main == module:
  print('Hello, World!')
```

---

This is a re-implementation of Node.py 0.0.22 and will be released as
version 0.1.0.
